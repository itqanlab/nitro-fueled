# Task Description - TASK_2026_018

## Problem

The previous hero animation was technically polished but visually generic. It looked like an enterprise AI particle/network demo:
- drifting points
- proximity links
- cursor aura

This did not express the product identity implied by the name **Nitro-Fueled**.

## Goal

Create a hero visual language that communicates speed, pressure, acceleration, and controlled energy while staying premium and readable for product messaging.

## Scope

- Keep page structure/content mostly intact
- Focus heavy changes on hero visual system
- Replace old canvas model with a directional fluid/streak model
- Improve hero layering/material treatment in CSS
- Keep static-file compatibility and reduced-motion support
